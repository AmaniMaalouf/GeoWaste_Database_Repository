The tool combines web scraping, LLM analysis, and reporting into one automated workflow.

### Workflow Overview

1. **Web Scraping**  
   Performs automated Google search based on user-provided dumpsite details (name, city, country). Extracts text content from relevant online sources using Selenium.

2. **Text Extraction & Cleaning**  
   Cleans raw HTML content and extracts meaningful text using BeautifulSoup. Compiles the extracted information into a structured format.

3. **PDF Report Generation**  
   Converts extracted text into a formatted PDF file (`extracted_texts.pdf`) using ReportLab.

4. **LLM-Powered Q&A Extraction**  
   Utilises OpenAI's GPT models (via LlamaIndex) to answer 37 structured questions about the dumpsite‚Äîcovering ownership, operations, environmental and social impact, and more. Outputs results in an Excel file.
